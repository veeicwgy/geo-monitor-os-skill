# Changelog

## 0.1.0 - 2026-04-17

Initial standalone release of the GEO monitoring Skill package.

### Added

- Publishable `SKILL.md` with semantic versioning and OpenClaw metadata.
- Discoverability-oriented naming and description for GEO monitoring, LLM monitoring, and developer-tool search intents.
- Bundled reference docs for workflow architecture, keyword strategy, monitoring system, content platform mapping, and negative-fix SOP.
- MinerU example case for demonstrating how the workflow applies to a real developer tool.
- Standalone `README.md` that explains positioning, install intent, and relationship to the full `geo-monitor-toolkit` repository.
